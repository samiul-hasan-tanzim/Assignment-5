1. What is the difference between var, let, and const?
    
        var → old way to make variable, can be change anywhere, function scoped.
        let → new way, can change, block scoped.
        const → cannot change, block scoped.
        So, var is old, let is new, const is for not change.

2. What is the spread operator (...)?

        Spread operator use to copy array or object. It make new array or object with same value. You can also use it to merge many array together. 
        Example: [...arr] make copy of arr.

3. What is the difference between map(), filter(), and forEach()?

        map() → change every item in array and return new array.
        filter() → check condition and return new array with items pass.
        forEach() → just do something for each item, no return.

4. What is an arrow function?

        Arrow function is short function. It use () => { } style. Many people like it because code looks clean and small.

5. What are template literals?

        Template literals is string with backtick `. You can write variable inside ${}. It make string easy and clean, no need + for join.
